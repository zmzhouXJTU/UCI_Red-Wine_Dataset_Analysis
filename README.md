## 红酒数据集分析


这个demo分析了红酒的**通用数据集**。这个数据集有**1599个样本，11个红酒的理化性质，以及红酒的品质（评分从0到10）**。

这里主要目的在于展示进行数据分析的常见Python包的调用，以及数据可视化。

---

### 主要内容分为：

* 单变量分析
  
* 双变量分析
  
* 多变量分析

---

### 数据集来源

经典的UCI数据集：<http://archive.ics.uci.edu/ml/datasets/Wine>

---

### 配置环境

* Python 3.6.5
* numpy 1.15.4
* pandas 0.23.4
* matplotlib 3.0.2
* seaborn 0.9.0
